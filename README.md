# FS25 Farm Reputation

A progression and reputation system for Farming Simulator 25.

## Features

- Reputation levels
- Reward bonuses
- Unlockable advantages
- Dynamic contract limits
- Multiplayer support

## Installation

1. Download the latest release.
2. Copy the ZIP file into your Farming Simulator 25 mods folder.
3. Start the game.

## Requirements

- Farming Simulator 25
- Latest game version
